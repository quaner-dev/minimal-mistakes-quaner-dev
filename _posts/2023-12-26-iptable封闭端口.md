# iptables封闭端口

## 阻止所有外部IP地址访问特定端口

```bash
iptables -A INPUT -p tcp --dport <端口号> -j DROP
```

## 允许特定IP地址访问端口

```bash
iptables -A INPUT -p tcp -s <IP地址1>,<IP地址2>,<IP地址3> --dport <端口号> -j ACCEPT
```

## 导出iptables规则

```bash
iptables-save > /root/iptables
```

## 保存iptables规则

```bash
iptables-restore < /root/iptables
```
